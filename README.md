# Sumiyo Analysis | AI Supply Chain Command Center

A high-fidelity dashboard for operational intelligence, supplier performance tracking, and AI-driven inventory decisions.

## 🚀 Live Project
View the dashboard here: [sumiyosheikh.com](https://sumiyosheikh.com)

## 🛠 The Tech Stack
* **Frontend:** HTML5, Tailwind CSS (Modern, dark-mode UI)
* **Backend:** Python (Server-side logic for data handling)
* **Data Integration:** Google Sheets API (Simulated ERP data feed)
* **Deployment:** GitHub Pages

## 📦 Core Modules
* **Supplier Risk Matrix:** Real-time scoring based on freight variance.
* **AI Assistant:** Inventory analysis and reorder recommendations.
* **Decision Support:** Critical alert system for supply chain disruptions.
